Welcome to flask-redis-multi documentation!
****************************************

.. image:: https://img.shields.io/pypi/v/flask-redis-multi.svg

flask-redis-multi is a flask extension package for communicating with a Redis server
with multiple databases.

Check out the source on `github <https://github.com/pe2mbs/flask-redis-multi.ix>`_.

Installation instructions
=========================

* Install Python 3.8 or newer.
* Install flask-redis-multi.

::
